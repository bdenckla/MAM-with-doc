# MAM-with-doc
Miqra According to the Masorah in HTML, with its documentation notes

This edition is deployed via the github-pages mechanism, at https://bdenckla.github.io/MAM-with-doc/.

The published sigil page at docs/sigil-decoding.html is manually maintained as a public-facing subset
of MAM-basics/doc/sigil-decoding.md. It is not generated automatically from the Markdown source, so
updates should be merged carefully rather than replacing the HTML wholesale.
